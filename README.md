# AjoVault Web

## Local start

Install dependencies:

```sh
npm install
```

Start the app on `http://localhost:8080`:

```sh
npm start
```

For the regular Vite development command, you can also use:

```sh
npm run dev
```

## Environment

Create a `.env` file from `.env.example` if needed:

```sh
VITE_API_BASE_URL=http://localhost:51708
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui
